# DataSense AI

An AI-powered data analytics assistant that enables users to upload datasets and gain insights through automated cleaning, visualization, anomaly detection, and predictive analytics.

## Features

- CSV & Excel Upload
- Dataset Preview
- Summary Statistics
- Data Cleaning
- Interactive Visualizations (Coming Soon)
- AI-Powered Insights (Coming Soon)

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
